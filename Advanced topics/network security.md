# 🔐 NETWORK SECURITY — FAANG LEVEL NOTES (DEEP + INTERVIEW READY)

---

# 🧠 1. WHAT IS NETWORK SECURITY?

👉 Network security is the practice of protecting networks and data from:
- Unauthorized access  
- Misuse  
- Cyberattacks  

---

# 🎯 CORE GOALS (VERY IMPORTANT ⭐)

👉 CIA TRIAD:

- **Confidentiality** → Only authorized users access data  
- **Integrity** → Data is not altered  
- **Availability** → Systems remain accessible  

---

# 🧠 2. HOW NETWORK SECURITY WORKS (LAYERED MODEL)

---

👉 Uses **Defense in Depth** (multiple layers of security)

---

# 🔐 3. TYPES OF SECURITY LAYERS

---

# 🌟 1. PHYSICAL SECURITY

---

## 🧠 PURPOSE

👉 Protect hardware from physical access

---

## 🔧 METHODS

- CCTV  
- Biometric access  
- Locked server rooms  

---

## 🧪 SCENARIO

👉 If attacker steals server → all data compromised  

---

# 🌟 2. TECHNICAL SECURITY

---

## 🧠 PURPOSE

👉 Protect data during:
- Transmission  
- Storage  

---

## 🔧 TOOLS

- Firewalls  
- Encryption  
- Antivirus  
- IDS/IPS  

---

## 🧪 SCENARIO

👉 Hacker tries to access server → firewall blocks  

---

# 🌟 3. ADMINISTRATIVE SECURITY

---

## 🧠 PURPOSE

👉 Define policies & access rules  

---

## 🔧 METHODS

- Authentication  
- Authorization  
- Role-based access  

---

## 🧪 SCENARIO

👉 Employee only accesses allowed data  

---

# 🔥 4. KEY NETWORK SECURITY CONTROLS

---

# 🌟 1. FIREWALL ⭐

---

## 🧠 FUNCTION

👉 Filters traffic based on rules  

---

## ⚡ TYPES

- Packet filtering  
- Stateful firewall  
- Next-gen firewall  

---

## 🧪 SCENARIO

👉 Blocks unauthorized port access  

---

# 🌟 2. IDS / IPS ⭐⭐

---

## 🧠 DIFFERENCE

- IDS → Detects  
- IPS → Detects + blocks  

---

## 🧪 SCENARIO

👉 Detects DDoS or brute-force attack  

---

# 🌟 3. ENCRYPTION

---

## 🧠 PURPOSE

👉 Converts data into unreadable format  

---

## 🧪 SCENARIO

👉 HTTPS uses encryption  

---

---

# 🌟 4. VPN ⭐

---

## 🧠 PURPOSE

👉 Secure remote access  

---

## 🧪 SCENARIO

👉 Employee accesses office network from home  

---

---

# 🌟 5. ACCESS CONTROL (NAC)

---

## 🧠 PURPOSE

👉 Only authorized users/devices allowed  

---

## 🧪 SCENARIO

👉 Unknown device blocked  

---

---

# 🌟 6. NETWORK SEGMENTATION

---

## 🧠 PURPOSE

👉 Divide network into smaller parts  

---

## ⚡ BENEFIT

👉 Limits attack spread  

---

## 🧪 SCENARIO

👉 Hack in one segment doesn’t affect others  

---
