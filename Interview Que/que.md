# 🚀 COMPUTER NETWORKS — FINAL TRICKY INTERVIEW QUESTIONS (WITH ANSWERS)

---

# 🧠 1. Why TCP is reliable?

👉 Uses:
- ACKs (acknowledgment)
- Retransmission
- Sequence numbers
- Flow + congestion control

---

# 🧠 2. TCP vs UDP (real difference)

👉 TCP:
- Reliable
- Ordered
- Connection-oriented

👉 UDP:
- Fast
- No guarantee
- Connectionless

---

# 🧠 3. Why HTTP is stateless?

👉 It does not store client info between requests

👉 Solved using:
- Cookies
- Sessions
- Tokens

---

# 🧠 4. What happens when packet is lost in TCP?

👉 Duplicate ACKs → Fast retransmit  
👉 cwnd reduced → congestion control  
👉 Severe case → timeout → slow start  

---

# 🧠 5. Why DNS is needed?

👉 Humans use names  
👉 Computers use IP addresses  

👉 DNS = name → IP mapping

---

# 🧠 6. What is ARP?

👉 Maps:
IP → MAC

👉 Used inside LAN

---

# 🧠 7. Switch vs Router

👉 Switch:
- Works on MAC (Layer 2)
- Inside LAN

👉 Router:
- Works on IP (Layer 3)
- Connects networks

---

# 🧠 8. What is NAT?

👉 Converts private IP → public IP

👉 Allows multiple devices to share one public IP

---

# 🧠 9. What is PAT?

👉 Many private IPs use one public IP with different ports

---

# 🧠 10. What happens if router has no route?

👉 Packet dropped  
👉 Or default route used  

---

# 🧠 11. What is 3-way handshake?

👉 Establish TCP connection:
- SYN
- SYN-ACK
- ACK

---

# 🧠 12. Why 4-way termination?

👉 Both sides close connection independently

---

# 🧠 13. What is cwnd?

👉 Congestion window → controls sending rate

---

# 🧠 14. What is slow start?

👉 Start small → grow exponentially

---

# 🧠 15. What is congestion avoidance?

👉 Grow slowly to avoid overload

---

# 🧠 16. What is Fast Retransmit?

👉 Retransmit after 3 duplicate ACKs

---

# 🧠 17. What is Fast Recovery?

👉 Reduce window → continue (no restart)

---

# 🧠 18. Cookie vs Session

👉 Cookie:
- Stored in browser

👉 Session:
- Stored on server

---

# 🧠 19. What is Token (JWT)?

👉 Stateless authentication method

---

# 🧠 20. 401 vs 403

👉 401 → Not authenticated  
👉 403 → Not authorized  

---

# 🧠 21. Why HTTPS?

👉 Adds encryption (TLS)

---

# 🧠 22. What is latency?

👉 Time delay in communication

---

# 🧠 23. What is bandwidth?

👉 Maximum data transfer rate

---

# 🧠 24. What is throughput?

👉 Actual data transferred

---

# 🧠 25. Why video buffers?

👉 Packet loss / congestion / high RTT

---

# 🧠 26. What is DNS cache?

👉 Stores previous DNS results to speed up lookup

---

# 🧠 27. What is MAC address?

👉 Unique hardware address of device

---

# 🧠 28. What is IP address?

👉 Logical address of device in network

---

# 🧠 29. What is subnetting?

👉 Dividing network into smaller networks

---

# 🧠 30. What is VLSM?

👉 Variable subnet sizes based on need

---
